# Password Strength Checker

A Python project that evaluates the strength of a password based on length, character variety, and common weak patterns. Built to practice Python fundamentals and demonstrate common security checks for passwords.

---

## Features
- Checks password length (minimum 8 characters).
- Validates presence of:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Detects common weak passwords (e.g., `password`, `123456`, `qwerty`).
- Returns a strength rating: **Weak**, **Medium**, or **Strong**.
- Suggests improvements for stronger security.
